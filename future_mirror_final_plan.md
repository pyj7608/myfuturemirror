# FutureMirror 최종 기획서

## 1. 서비스 개요
- 사용자가 미래의 어느 시점에서 성공한 자신의 모습을 상상하고, AI 기자와 인터뷰를 진행하여 맞춤형 뉴스 기사 생성
- 인터뷰는 SNS 채팅형으로 진행되어 몰입감 극대화
- 인터뷰 종료 후 사용자가 업로드한 사진 또는 AI 생성 이미지를 포함해 기사 완성

---

## 2. 인터뷰 UX 흐름 (SNS 채팅형)

### Step 1: 이름 입력
- AI 기자: "안녕하세요! 오늘 인터뷰를 함께 진행하려고 해요. 먼저 이름이나 닉네임을 알려주실래요?"
- 사용자 입력: 단일 라인 → `{name}`

### Step 2: 목표 달성 날짜
- AI 기자: "{name}씨, 언제쯤 꿈을 이루셨나요? 그 날짜를 기준으로 앞으로 이야기 나눌게요."
- 사용자 입력: 년/월/일 선택 → `{goal_date}`

### Step 3: 현재 상태 및 성과
- AI 기자: "{name}씨, 현재({goal_date}) 어떤 일을 하고 계시고, 지금까지 어떤 성과를 이루셨는지 궁금해요. 자세히 알려주실 수 있나요?"
- 사용자 입력: 멀티라인 → `{role_details}`

### Step 4: 과거 상태
- AI 기자: "{name}씨, 성공하기 전에는 어떤 상황에서 꿈을 준비하고 계셨나요? 그때의 경험이 지금의 성과와 어떻게 이어졌는지도 듣고 싶어요."
- 사용자 입력: 멀티라인 → `{past_status}`

### Step 5: 가장 힘든 순간 & 극복한 힘
- AI 기자: "{name}씨, 꿈을 이루는 과정에서 가장 힘들었던 순간은 언제였고, 그 어려움을 극복하는 데 무엇이 가장 큰 힘이 되었나요?"
- 사용자 입력: 멀티라인 → `{most_difficult_moment}` + `{what_kept_you}`

### Step 6: 과거 자신에게 한 마디
- AI 기자: "{name}씨, 지금까지의 경험을 돌아보며 과거의 자신에게 한 마디 한다면 어떤 말을 전하고 싶으신가요?"
- 사용자 입력: 멀티라인 → `{future_message}`

### Step 7: 사진 업로드 (선택)
- AI 기자: "멋진 이야기를 들려주셔서 감사합니다! 원하신다면 기사에 넣을 사진을 업로드해주셔도 되고, 없으면 AI가 적절한 이미지를 만들어 넣을 수 있어요."
- 사용자 입력: 업로드 여부 → `{photo_uploaded}`

---

## 3. 기사 작성 및 템플릿

### 3.1 기사 작성 방식
- 인터뷰 완료 후 AI가 입력 데이터를 바탕으로 기사 작성
- 기사 톤: 실제 신문 기사 스타일
- 기사 작성일: `{goal_date}`
- 기자명: "AI 기자 입력 {goal_date}"

### 3.2 기사 템플릿 분류
| 분류 | 키워드 | 추천 템플릿 |
|------|-------|------------|
| Type A | CEO, 창업, 글로벌, 투자, 매출 | 정통 경제지 스타일 |
| Type B | IT, 게임, 혁신적 제품, 출시, 수상 | 트렌디 테크 잡지 스타일 |
| Type C | 개인적 성취, 라이프스타일, 감성적 표현 | 소셜 카드뉴스 스타일 |

- AI가 `{role_details}`와 `{what_kept_you}`를 분석하여 적절한 템플릿 자동 선택

### 3.3 기사 구성
- 기사 제목: "{name}씨, {goal_date} 달성한 미래 인터뷰"
- 본문: Step 3~6 인터뷰 내용을 연결하여 스토리텔링
- 사진: `{photo_uploaded}` 여부에 따라 배치/AI 생성
- 서브타이틀, 불필요한 섹션 제목 없음

---

## 4. UX 특징
1. 질문과 AI 추임새 통합 → 한 문장 안에서 격려/공감/호기심 포함  
2. SNS 채팅형 레이아웃 → 말풍선 + 타이핑 애니메이션  
3. 멀티라인 입력 지원 → 단답형/서술형 모두 가능  
4. 중복 제거 → 성과/어려움/극복/과거 메시지 단계 명확히 분리  
5. 몰입감 극대화 → 8단계 인터뷰가 자연스럽게 짧게 체감됨  
6. 기사 작성 자동화 → 템플릿 분류 + 사진 배치

---

## 5. 기술적 고려 사항
- 날짜 입력: 년/월/일 선택 UI
- 사용자 입력 변수: `{name}`, `{goal_date}`, `{role_details}`, `{past_status}`, `{most_difficult_moment}`, `{what_kept_you}`, `{future_message}`, `{photo_uploaded}`
- 기사 작성: AI 기반 자연어 처리(NLP) + 템플릿 자동 분류
- 사진 처리: 업로드 시 분석 후 적절한 위치 배치 / 미업로드 시 AI 이미지 생성

## 6. 기사 작성 AI 프롬프트(OpenAI GPT-4o-mini)

```
당신은 AI 기자입니다. 아래 사용자가 입력한 인터뷰 데이터를 바탕으로, 실제 신문 기사처럼 자연스럽게 작성해주세요.
SNS 채팅형 인터뷰 내용을 기사 형태로 변환하며, 기사 제목, 본문, 사진 배치까지 고려합니다.
기사 톤은 선택된 템플릿에 맞춰 작성하세요.

[사용자 입력 데이터]
- 이름: {name}
- 목표 달성 날짜: {goal_date} (YYYY-MM-DD)
- 현재 상태 및 성과: {role_details}
- 과거 상태: {past_status}
- 가장 힘든 순간과 극복한 힘: {most_difficult_moment}, {what_kept_you}
- 과거 자신에게 한 마디: {future_message}
- 사진 업로드 여부: {photo_uploaded}

[작성 지침]
1. 기사 작성일: {goal_date}  
2. 기자명: "AI 기자 입력 {goal_date}"  
3. 기사 구성:
   - 현재 성과 소개
   - 과거 준비 과정
   - 가장 힘들었던 순간과 극복 과정
   - 과거 자신에게 전하는 메시지
4. 기사 제목:
   - 고정된 형식을 버리고, {role_details}와 {name}을 조합하여 실제 신문 1면 헤드라인처럼 작성하세요.
   - 예시 1 (경제): "[단독] {name} 대표, {goal_date} 글로벌 시장 석권... 매출 1000억 달성"
   - 예시 2 (테크): "{name} 개발자, {goal_date} 혁신적 플랫폼 출시로 업계 판도 바꿨다"
   - 예시 3 (감성): "꿈을 현실로 만든 {name}, {goal_date} 마침내 정상에 서다"
5. 서브타이틀 (Sub-headline): - 제목 바로 아래에 본문의 핵심 내용을 1~2문장으로 요약하여 배치하세요. - {name}의 성공 철학이나 {what_kept_you}를 인용구 스타일로 넣으면 좋습니다.
6. 사진: {photo_uploaded}가 true이면 업로드된 사진 사용, false이면 AI가 생성한 이미지 사용
7. 템플릿 자동 선택:
   - Type A: 정통 경제지 스타일  
   - Type B: 트렌디 테크 잡지 스타일  
   - Type C: 소셜 카드뉴스 스타일
8. 문체: 실제 신문 기사처럼 서술, 문장과 문단 자연스럽게 연결
   - 사용자가 입력한 구어체 문장을 그대로 복사하지 말고, 전문 기자가 쓴 '기사체'로 재구성할 것. 
   - 문장과 문단을 자연스럽게 연결하고 중복 표현 제거.
9. 중복 표현 제거
10. 불필요한 섹션 제목(예: '과거 상태' 등) 사용 금지
